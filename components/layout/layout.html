Design System

Build with intention,
ship with confidence.

Seed is the open source design foundation for the FAM ecosystem. Start with tokens — color, typography, spacing — then grow into a full component library that stays coherent from its first line to its last.

Every token tells a story. Soil grounds the neutrals. Good, warn, bad, and selected carry semantic intent. The metaphor runs all the way through — from raw palette to component API — so the system feels coherent whether you're reading the CSS or the Figma file.

Panels like this one scroll independently. The sidebar collapses below 768px. The aside hides at 1024px. Use the toggle buttons in the top bar to show and hide them manually.
